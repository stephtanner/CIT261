Javascript: Basic Definition and Uses

Javascript is used to make a web page more interactive.

Access Content

Select any element, attribute, or text from an HTML page

Modify Content

Add elements, attributes, and text to the page, or remove them

Program Rules

a recipe for the computer to change things on the page

e.g. a gallery that enlarges and image when you click on it

React to Events

have a script run... on click, on hover, after 10 seconds, on scroll, etc.

Javascript Variables

containers for storing data values

declare using var keyword

All variables must have unique names within their scope:

  • In a function, variables are local, meaning they are only inside the function
  • Global Variables are created outside a function (and take up more memory)

var x = 9;

var y = 7;

var z = x + y;

document.getElementById("practice").innerHTML = z;

display answer: 16

Javascript Functions

A group of statements to perform a specific task. You can reuse a function rather than repeat the code.

Declaring a function: function functionName() {...

Calling a function: functionName();

Javascript Arrays

A special type of variable that stores a list of values

You should use an array whenever you are working with a list or a set of values that are related

You do not need to specify how many items an array will hold

Example

var colors;
colors = ["white", "yellow", "orange"];
