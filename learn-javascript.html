Javascript: Basic Definition and Uses

Javascript is used to make a web page more interactive.

Access Content

Select any element, attribute, or text from an HTML page

Modify Content

Add elements, attributes, and text to the page, or remove them

Program Rules

a recipe for the computer to change things on the page

e.g. a gallery that enlarges and image when you click on it

React to Events

have a script run... on click, on hover, after 10 seconds, on scroll, etc.

Javascript Functions

A group of statements to perform a specific task. You can reuse a function rather than repeat the code.

Javascript Variables

containers for storing data values
